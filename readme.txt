To use the Frag_finder_app.ipynb Jupyter Notebook clone the repository using:

git clone https://github.com/jaredraynes/fragment_finder.git

Install the Jupyter Notebook environment by:

Using anaconda prompt, move to the fragment_finder directory and type:

conda env create -f environment.yml

To see that the new environment was installed correctly type:

conda env list

To activate the environment type:

conda active fragment_finder-env

Now Run Jupyter Notebook by typing:

Jupyter Notebook

Open the Frag_finder_app Notebook and Run all cells
